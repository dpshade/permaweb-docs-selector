DocSelector Component Example

Basic Usage

The DocSelector component appears in the bottom-right corner. It automatically detects your site's theme and adapts accordingly.

Method 1: Auto-Detection (Recommended)

The component automatically detects which site it's on based on the URL:

// No configuration needed - just load the script!
<script src="https://arweave.net/YOUR_TRANSACTION_ID"></script>

// Auto-detection patterns:
// ‚Ä¢ AO: URLs with 'cookbook_ao', 'ao.', or '/ao/'
// ‚Ä¢ Hyperbeam: URLs with 'hyperbeam' or '/hyperbeam/'  
// ‚Ä¢ Arweave: URLs with 'cookbook', 'arweave', '/cookbook/' (default)

Method 2: Manual Override (Optional)

Force a specific configuration if auto-detection doesn't work:

// Override auto-detection (optional)
window.DocSelectorConfig = {
    currentCookbook: "ARWEAVE", // Force: AO, HYPERBEAM, or ARWEAVE
    links: {
        AO: "https://cookbook_ao.arweave.net/",
        HYPERBEAM: "https://hyperbeam.arweave.net/",
        ARWEAVE: "https://cookbook.arweave.net/"
    }
};

        // Load the script
        <script src="../component/doc-selector.js"></script>

Configuration Options

  • currentCookbook: The currently active documentation site ("AO", "HYPERBEAM", or "ARWEAVE")
  • links: Object containing the URLs for each documentation site
  • theme.mode: Theme mode - "auto" (default), "light", or "dark"
  • theme.colors: Custom colors for light and dark themes

Theme Configuration Examples

// Force dark theme
window.DocSelectorConfig = {
    theme: { mode: "dark" }
};

// Force light theme  
window.DocSelectorConfig = {
    theme: { mode: "light" }
};

// Auto theme with custom colors
window.DocSelectorConfig = {
    theme: {
        mode: "auto",
        colors: {
            light: { text: "#2d3748", line: "#4a5568", background: "#f7fafc" },
            dark: { text: "#f7fafc", line: "#cbd5e0", background: "#1a202c" }
        }
    }
};
