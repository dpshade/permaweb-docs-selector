DocSelector Component Example

Basic Usage

The DocSelector component appears in the bottom-right corner. It automatically detects your site's theme and adapts accordingly.

Method 1: Use Built-in Constants (Recommended)

Modify the constants directly in the source code:

// In dist/doc-selector.js - modify these constants:
const DEFAULT_CURRENT_COOKBOOK = "ARWEAVE"; // Change for each site
const DEFAULT_DOCUMENTATION_LINKS = {
    AO: "https://cookbook_ao.arweave.net/",
    HYPERBEAM: "https://hyperbeam.arweave.net/",
    ARWEAVE: "https://cookbook.arweave.net/"
};

// Then simply load:
<script src="https://arweave.net/YOUR_TRANSACTION_ID"></script>

Method 2: Runtime Configuration

Override defaults at runtime (useful for dynamic setups):

// Configure before loading the script
window.DocSelectorConfig = {
    currentCookbook: "ARWEAVE", // AO, HYPERBEAM, or ARWEAVE
    links: {
        AO: "https://cookbook_ao.arweave.net/",
        HYPERBEAM: "https://hyperbeam.arweave.net/",
        ARWEAVE: "https://cookbook.arweave.net/"
    }
};

// Load the script
<script src="../dist/doc-selector.js"></script>

Configuration Options

  • currentCookbook: The currently active documentation site ("AO", "HYPERBEAM", or "ARWEAVE")
  • links: Object containing the URLs for each documentation site
